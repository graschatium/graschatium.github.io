FAQ - Foire Aux Questions

Retrouvez ici les réponses aux questions les plus fréquentes sur Graschatium.

Comment rejoindre le serveur Graschatium ?

Pour rejoindre Graschatium, téléchargez le launcher depuis notre site officiel, installez-le, et connectez-vous avec votre compte Minecraft Java.

Quels sont les prérequis pour jouer sur Graschatium ?

Vous devez posséder un compte Minecraft Java Edition et un ordinateur sous Windows ou macOS. Les performances peuvent varier en fonction des configurations matérielles.

Graschatium est-il gratuit ?

Oui, le serveur est totalement gratuit. Certaines fonctionnalités optionnelles ou objets cosmétiques peuvent être disponibles à l'achat dans notre boutique en ligne.

Que faire si je rencontre un bug ?

Si vous rencontrez un problème, vous pouvez le signaler en remplissant le formulaire dédié sur la page des rapports de bugs.

Comment rejoindre l'équipe du serveur (staff) ?

Pour rejoindre notre équipe, consultez la page recrutements. Les informations et critères requis y sont listés.
